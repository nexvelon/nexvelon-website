# Nexvelon Global — Marketing Website

Production website for Nexvelon Global (nexvelonglobal.com).

**Stack:** Static HTML, deployed via Vercel from this repo.

**Files:**
- `index.html` — Main site
- `images/` — Brand imagery (WebP)
- `sitemap.xml` — Google sitemap
- `robots.txt` — Crawler instructions
- `llms.txt` — AI crawler optimization
